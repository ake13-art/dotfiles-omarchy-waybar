# waybar dotfiles (for omarchy)
---
## waybar dotfiles with:
- preset omarchy and arch icon (selectable in config.jsonc)
- hyprland/workspace as pacman and his colorful ghosts
- script that displays upload and download speeds
- time and icon that shows updates (under Omarchy)
- Symbols for Wi-Fi, Bluetooth, and sound
- Overview of CPU temperature, utilization, and disk usage
- Battery indicator with percentage display
---
## "Installation"
- Download script/ and place it in ~/.config/waybar/ (or if ~/.config/waybar/script/ already exists: add net-speed.sh to this folder).
- Replace local config.jsonc and style.css
